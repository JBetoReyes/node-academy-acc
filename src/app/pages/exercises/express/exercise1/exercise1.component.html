Exercise 1

Requirements

  • Create an express app and a new server using the port 3000
  • Create a new endpoint
  • The new endpoint should return a set of mock customers
  • The response should come in JSON format
  • The Content/Type header should be application/json

Setting our app with express

  • Create a new file called app.js with the following content
> app.js ```javascript const express = require('express'); const chalk = require('chalk'); const app = express(); app.get('/customers', (req, res) => res.json([ {{ '{' }} name: 'John', lastName: 'Doe', email: 'Doe@gmail.com' {{ '}' }}, {{ '{' }} name: 'Jose', lastName: 'Perez', email: 'Jose@gmail.com' {{ '}' }} ])) module.exports = app; ```
  • You can test your endpoint using the following embed runkit
  • Lets create our web server
> bin/www ```javascript const app = require(../app); const port = 3000; app.listen(port, () => {{ '{' }} console.log('Listening in port ' + port); {{ '}' }}); ```
