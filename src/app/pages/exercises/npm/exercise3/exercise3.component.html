Exercise 3

Requirements
  • Create new function in your module called searchBy
  • This function should help you to find logs by a criteria
  • The first argument should be the name and path of your log file; ie. my/path/log.txt
  • The second argument of your function should be the criteria of search
  • The function should be able to receive two kinds of criteria: Text, Time
  • The function should return an array filled with the found log messages
  • If no log is found the function should return an empty array
